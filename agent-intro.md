## 一句话看懂区别

| 维度    | 普通大模型问答           | Claude Code / Qwen Code（AI 编程助手）        |
| ----- | ----------------- | --------------------------------------- |
| 交互位置  | 浏览器/IM 里“贴代码”     | 直接嵌在终端/IDE，**当前代码库就是上下文**               |
| 上下文长度 | 你手动挑 4k-32k token | 自动索引**整仓库**（百万 token），跨文件推理             |
| 工具调用     | 0 个，只能“给答案”                   | 内置 10+ 工具链：**bash、git、gdb、make、docker、kubectl、pytest、perf、tcpdump……** |
| 多步闭环     | 人驱动每一步                        | 自己规划：改代码 → 编译 → 跑单测 → 定位失败行 → 再打 patch                                |
| Debug 循环 | 你贴报错，它猜原因                     | 它**主动跑复现脚本**，用 gdb 捞寄存器，对比 core 文件，再改                                 |
| 部署验证     | 你手动 `scp + systemctl restart` | Agent 一键 `docker build → helm upgrade → 流量灰度 → 看 metrics`             |

## 使用案例
| 场景                         | 传统做法                         | 耗时        | AI 编程助手做法                                                                                  | 耗时     | 节省  |
| -------------------------- | ---------------------------- | --------- | ------------------------------------------------------------------------------------------ | ------ | --- |
| 读开源 RocksDB/PostgreSQL 新特性 | `git log -p` 人肉翻 200  commit | 2-4 h     | `qwen-code why was `WriteBatch` internal rep changed in 8.10?` 自动读全历史 → 生成 5 行摘要 + 关键 diff | 2 min  | 95% |
| 给 40 个存储层接口加统计计数           | 手动改头文件+源文件+注册宏               | 1 d       | 一条指令批量补全，自动把宏、头文件、初始化函数改好 → 直接 `make -j`                                                   | 10 min | 90% |
| 写并发压力测试 driver             | 从上游 copy 一份 `gtest`，再改参数、改断言 | 1-2 h     | 描述“模拟 100 线程随机读写，验证快照隔离” → 自动生成 `folly::Benchmark` 框架 + 校验逻辑                               | 5 min  | 92% |
| 追查 core dump，看不懂栈          | `gdb` 来回 `list`，查结构体字段       | 30-60 min | 把 `bt` + 寄存器丢给 AI，自动标注每个变量在源码位置、可能空指针分支                                                    | 3 min  | 90% |
| 发版前整理 commit message       | 人肉拼 50 条 `git log`           | 30 min    | `qwen-code generate release note since v7.2` → 按模块归类，挑 PR 标题，自动生成 markdown                 | 1 min  | 97% |
